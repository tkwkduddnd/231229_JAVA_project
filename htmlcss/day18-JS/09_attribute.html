element 속성은 크게 두가지로 구분 - 태그의 속성 - css(style) 속성

  • 태그의 속성은 태그가 갖는 속성이기 때문에 속성명을 그대로 사용 tagName.id, tagName.src,tagName.href ...
  • class처럼 여러개의 값을 갖는 속성들은 별도의 속성명을 사용 tagName.classList
  • css 관련 속성들은 style.을 통해 프로퍼티에 접근 가능 tagName.style.color
  • css 속성 중 font-size => - 를 지우고 첫 글자를 대문자로 변경 fontSize
