window 메서드 종류

  • window.open('url', 'window name', 'option(size, scroll...)');
  • window.close(); : 창 닫기
  • alert(); / prompt(); input 포함 / confirm(); 확인/취소 창
  • setInterval(function, millsecond); 일정 간격 지속적으로 실행문을 실행시킬 때 사용
  • setTimeout(function, millsecond); 일정 시간동안 실행문을 실행시킬 때 사용
