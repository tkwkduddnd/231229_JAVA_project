반복문 : for, while, do~while => java와 동일한 문법

  • for : 반복 횟수가 명확할 때 사용 for(초기값; 조건식; 증감식) {코드....;}
  • while, do~while : 반복횟수가 명확하지 않을 때 사용 while(조건식) {실행문...;} do{실행문...;} while(조건식);
  • 모든 조건은 참(true)일때 실행
  • 조건이 계속 true라면 무한루프에 빠질 수 있음 조건문 안에서 조건값이 false로 바뀔수 있는 구문이 반드시 있어야함
  • false를 줄 구문이 없다면 break;를 통해 빠져나갈 수 있음
