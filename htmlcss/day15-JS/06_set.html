자바 스크립트의 데이터 타입의 확장 개념 집합 데이터를 다룰 수 있는 객체

  • set 중복데이터 x : 중복 데이터가 들어오면 체크 후 하나만 남김
  • set() : 파라미터로 주로 배열을 사용, index, value 값을 활용하여 집합데이터 생성
  • new Set() or Set(array) : set 객체를 생성, 초기화
  • set.add(value) : 데이터 추가
  • set.delete(value) : 데이터 삭제
  • set.has(value) : 값의 존재 여부를 true / false로 리턴
  • set.forEach(callback function) : set의 순환
  • set.entries(), set.keys(), set.values() 리턴타입 setIterator 형식임.
  • set.size : set size 리턴
  • set.clear() : set 비우기
