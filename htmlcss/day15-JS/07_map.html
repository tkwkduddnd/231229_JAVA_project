Map : key : value 형태의 데이터를 한개의 배열로 만들어서 다시 배열에 저장하는 방식

  • new Map(), new Map([[key,value], [key,value], [key,value]]);
  • 기본적으로 중첩 구조를 가지고 있기 때문에 key 중복확인 필수
  • key를 식별자로 하는 get(keyName), set(keyName, value) 메서드를 제공
  • 2차원 배열과 비슷한 구조
  • key는 문자로 주는것을 원칙으로 함.
  • get(keyName) : key에 해당하는 value값을 리턴
  • set(keyName, value) : key에 해당하는 value를 변경
  • delete() / has() / forEach() / keys() / values() / entries() / size() / clear() 전부 가능
